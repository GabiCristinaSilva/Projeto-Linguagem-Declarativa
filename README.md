# Paradigmas em Phyton
a)Linguagens imperativas descrevem como o computador deve fazer uma tarefa, mostrando passo a passo as instruções. Já linguagens declarativas (a do projeto feito)  dizem apenas o que deve ser feito, focando no resultado sem detalhar o processo. Em resumo, imperativa é o “como fazer” e declarativa é o “o que fazer”.
b)Sendo o site uma linguagem interpretada pois traduzem linha por linha em tempo real.
c)A forma mais comum de executar o html é no servidor web. Basta localizar o arquivo salvo com a extensão .html e dar um duplo clique nele, ou arrastá-lo para dentro de uma janela do navegador.
d)O HTML não tem variáveis como linguagens de programação.
e)é uma linguagem de marcação (não de programação) e, portanto, não possui tipagem forte ou fraca.
